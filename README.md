# Simple Egg Boil App

App to time the cooking of soft, jammy, medium, or hard boiled eggs. 🥚🐣

Original Art with Figma ✨

## Features
- Egg selector with clickable SVGs
- Timer screen
- Responsive layout with Tailwind CSS

## Tech Stack
- React
- Vite
- Tailwind CSS
- Figma
