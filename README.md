# Simple Egg Boil App

A React + Vite + Tailwind app to select and time for soft, jammy, medium, or hard boiled eggs. 🥚🐣
Original Art with Figma ✨

## Features
- Egg selector with clickable SVGs
- Timer screen
- Responsive layout with Tailwind CSS

## Tech Stack
- React
- Vite
- Tailwind CSS
- Figma
